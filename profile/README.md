# Relumic

**Edge tools for defense, government, and critical infrastructure.**

We build open source software for environments where reliability isn't optional — unreliable networks, constrained hardware, air-gapped deployments, and operational contexts where a failure isn't a ticket, it's a mission impact.

---

## What We're Building

### [Relay](https://github.com/relumic/relay) — *in development*
Priority-ordered data sync across an unreliable connection. When a bandwidth window opens, the highest-priority payload moves first — every time. WAL-backed persistence, gRPC transport, single static binary. No runtime dependencies.

---

## Principles

- **Correctness before features.** Every guarantee we make must be real.
- **Deployability is non-negotiable.** Air-gapped, 2 GB RAM, no separate processes. If it doesn't run there, it isn't finished.
- **Code is a trust artifact.** Defense engineers read the source before they adopt the tool.
- **Open source foundation.** The infrastructure others build on must be open and auditable.
